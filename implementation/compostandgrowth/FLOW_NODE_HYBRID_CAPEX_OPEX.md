# FLOW NODE HYBRID – CAPEX & OPEX (EN)

## Overview
This document provides a detailed cost analysis for the three types of Flow nodes:
- Urban Node (500 m²)
- Peri-Urban Node (2 ha)
- Regional Node (1 000 m², high-energy)

Costs are divided into:
- **CAPEX**: Equipment, installation, infrastructure
- **OPEX**: Energy, maintenance, consumables, labor equivalents

## Notes

1. All figures are **approximate** and based on currently available commercial technology.
2. Energy production from solar/wind is based on realistic daily averages.
3. Labor equivalents use $10/h valuation for comparison, but Flow uses Flow-hours internally.
4. Automation levels:
   - Level 2: Semi-automated irrigation, monitoring
   - Level 3: Robotized harvest, automated climate control
   - Level 4: Full automation including planting, nutrient dosing, monitoring
