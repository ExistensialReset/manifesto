## 🌊 Flow Node Water Filter — Quick Cheat Sheet

```text
        ┌───────────────┐
        │ Funnel / Neck │  ← directs water into filter
        └───────┬───────┘
                │
                ▼
        ┌───────────────┐
        │ Cloth / Cotton│  ← stops fine debris, holds sand
        └───────┬───────┘
                │
                ▼
        ┌───────────────┐
        │  Fine Sand    │  ← removes small particles
        └───────┬───────┘
                │
                ▼
        ┌───────────────┐
        │ Coarse Sand   │  ← supports fine sand, slows flow
        └───────┬───────┘
                │
                ▼
        ┌───────────────┐
        │ Gravel / Stones│ ← prevents clogging, distributes water
        └───────┬───────┘
                │
                ▼
        ┌───────────────┐
        │  Collection   │  ← clean water gathers here
        └───────┬───────┘
                │
   ┌────────────┴────────────┐
   │ Optional Disinfection:   │
   │ 1. Boiling ≥1 min        │
   │ 2. SODIS 6h+ sunlight   │
   └─────────────────────────┘
```

**💡 Layer Tips**
- Funnel / Neck: Direct flow, avoid splashing or overflow.
- Cloth / Cotton: Replace if clogged or dirty.
- Fine Sand: Rinse before use, keep dry before layering.
- Coarse Sand: Improves flow and filtration efficiency.
- Gravel / Stones: Prevents channeling, evens out water spread.
- Collection: Store in clean container; avoid touching filtered water.
- Disinfection: Always boil or SODIS water for microbial safety.

**⚡ Quick Notes**
- Uses only locally available materials.
- Fully replicable in Flow Node Circles.
- Emphasizes Local Capacity Sufficiency: anyone with energy, knowledge, and basic tools can implement.
- Designed for safety, simplicity, and community resilience.
