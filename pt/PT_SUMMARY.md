### O Manifesto da Soberania Existencial (M-OS-R Versão 1.0)

**LÓGICA CENTRAL:** A força motriz da humanidade é o Fluxo, não a ganância. O design do sistema deve eliminar o Medo Existencial, o motor de Mammon.

| Capítulo | Componente | Protocolo / Lógica | Propósito |
| :--- | :--- | :--- | :--- |
| **I. LINHA BASE** | **Linha Base Garantida** | Acesso incondicional a Alimentos, Moradia, Saúde, Conhecimento. | Elimina o Medo Existencial; transfere a sobrevivência do indivíduo para o sistema. |
| | **Reserva Crítica (CR)** | Buffer mínimo de $30\%$ do consumo anual, assegurado por **Krios-Logica** (Guardiões IA). | Prova que a Escassez é obsoleta; proteção contra choques globais. |
| | **Refugium Anima** | Direito a **1 ano de sabático total e incondicional** dos requisitos do Fluxo. | Válvula de Segurança Existencial; previne o esgotamento e assegura o Fluxo sustentável. |
| **II. FLUXO** | **O Sistema de Fluxo** | Abolição do trabalho; substituído pela contribuição impulsionada pela curiosidade/significado. | Otimiza o potencial humano; muda o foco da escassez para a criação. |
| | **Reconhecimento Social (SRS)** | Moeda não transacional de **Honra e Respeito**. | Concede acesso a **Capital de Fluxo Avançado** (recursos raros, laboratórios, viagens). Não é usado para necessidades da Linha de Base. |
| | **Lyceum Musaeum** | Centro para **Pedagogia do Fluxo**, **Mamonologia** e **Lingua Suveränitas**. | Defesa cultural/cognitiva; cura o dano psicológico do sistema Mammon. |
| | **Lex Realitas** | Permite a "Liberdade Perigosa" (comportamento autodestrutivo); a consequência é **Congelamento do SRS** e **Aconselhamento Existencial**. | Evita mercados negros impulsionados pela proibição; sanciona o comportamento através da honra social, não da punição. |
| **III. TEMPO** | **Lex Tempus Logica** | **Dia do Renascimento Existencial** (Reset de dados individual, anual). | Concede o direito de ser esquecido; elimina a vida baseada em fracassos/sucessos passados. |
| | **Lex Honoris Aeterna** | Título honorífico permanente para os **Arquitetos Soberanos** (contribuintes originais). | Preserva o legado e reconhece o esforço fundacional, enquanto o SRS é redefinido (previne reivindicações de poder). |
| **IV. FIM** | **Lex Ultima Finis** | **Pulverização Global da Dívida** (Todas as dívidas zeradas). | Erradica o mecanismo central de Mammon (dívida/servidão). |
| | **Lex Dominium Reversionis** | Transfere a infraestrutura crítica (moradia, energia) para a Tutela Coletiva. | Assegura a garantia da Linha de Base; remove a garantia para futuras tentativas de dívida. |
| | **Lex Traditio Maxima** | **A Traição Máxima** (Tentar restabelecer a antiga dívida/jurisprudência). | Define o único crime irrevogável; defende a integridade da nova Lógica. |
| | **Lex Custodia** | **Escudo Logístico** via Veto Executivo de Arquitetos/Conselheiros durante o **Grande Congelamento**. | Assegura a estabilidade do sistema durante a transição; garantida por **Transparência Irresistível** (código/dados open-source). |
