## 🌐 Usage Notes

- All files are **living documentation**. Updates should always include versioning and audit trails.  
- LOTUS, Nodes, and Regional/Global governance must **adhere strictly to protocols** defined in `LOTUS_PROTOCOL.md`.  
- Emergency procedures (under attack) are **separate but integrated** via `BODY_OF_FLOW_UNDER_ATTACK.md`.  
- Every operational or structural change should reference **related docs** to maintain integrity.
