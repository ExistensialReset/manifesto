# 1. HEART — `/ethos`

**Function:** Regulates moral temperature.  
**Failure mode prevented:** Coercion drift.

`/ethos` defines:

- Voluntary participation  
- Compost & Growth principle  
- Mistakes without exile  
- Non-performative value  
- Anti-dominance orientation  

If `/ethos` collapses:

The system becomes administrative.  
Administrative systems drift toward control.  

Therefore:

`/ethos` must remain socially reinforced, not procedurally enforced.

It cannot be replaced by policy.

---

# 2. NERVOUS SYSTEM — `/core`

**Function:** Signals structural reality.  
**Failure mode prevented:** Expansion beyond capacity.

`/core` defines:

- Resource measurement standards  
- Baseline invariants  
- Aggregated metrics (never individual scoring)  
- Pause / Shrink / Fork / End logic  

The Baseline invariant is the primary reflex:

If minimum ecological + human stability cannot be maintained →  
The system must pause or contract.

This reflex prevents imperial drift.

The system expands only under surplus stability.  
It contracts under strain.

---

# 3. SKELETON — `/structure_in_flow`

**Function:** Provides scalable structure without rigid hierarchy.  
**Failure mode prevented:** Centralized consolidation.

Structural layers:

- **Micro-Circle** — relational cell  
- **Baseline Circle** — stabilized cluster  
- **Flow Node** — functional organ  
- **Regional Network** — federated organs  
- **Global Layer** — rare resource coordination  

Each level:

- Emerges only after demonstrated stability  
- Can contract independently  
- Does not override lower-level autonomy  

Authority flows downward.  
Coordination flows upward.

No layer has permanent command rights over another.

---

# 4. METABOLISM — FLOW ACCOUNTING

**Function:** Circulates resources without monetization.  
**Failure mode prevented:** Individual performance stratification.

Principles:

- No currency layer  
- No individual tracking  
- Aggregated contribution visibility  
- Capacity-based allocation  

Flow measures systemic health, not human worth.

Metrics are diagnostic, not moral.

Includes:

- Baseline Emergency Bypass  
- Infrastructure Sabotage Protocol  
- Conflict Mediation Layers

Purpose:

- Detect instability early

- Minimal

- Reviewable

Based on `FLOW_SPIRAL_REALISTIC_TIMELINE_REVIEW`.

# 7. CONTRACTION — LEAF FALL MECHANISM

Critical safeguard:

Any layer may shrink without stigma.

Contraction is:

- Structural hygiene  
- Burnout prevention  
- Anti-expansionist reflex  
- Ecological realism  

Systems that cannot shrink collapse.

M-OS-R encodes shrinkage as strength.

Dissolution is not failure.

It is compost.
