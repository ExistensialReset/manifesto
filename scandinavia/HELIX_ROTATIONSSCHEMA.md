**Status:** Draft  
**Author:** Elinor Frejd / ChatGPT 
**Date:** 2026-01-28  

---

## 1. Översikt

- 29 medlemmar → 4 huvudområden + Refugium Anima  
- Uppgifter delas upp i **veckor** med roterande ansvar  
- Alla uppgifter märks med:  
  - Typ: Fysisk / Administrativ / Social / Kreativ  
  - Tidsåtgång: Kort (1–2 h), Medium (2–4 h), Lång (>4 h)  
  - Frekvens: Daglig, Veckovis, Månatlig  

---

## 2. Huvudområden

| Område | Uppgifter | Kommentar |
|--------|-----------|-----------|
| **Odling & Skörd** | Vattning, plantering, skörd, packning | Växter, bär, örter, vertikala odlingar |
| **Ocean-system** | Foder, vattenkontroll, rengöring, hälsokoll | Akvarier, musslor, bläckfiskar, tilapia |
| **Refugium Anima** | Värd, städning, återställning, materialpåfyllnad | 2–3 personer/vecka, roterande |
| **Gemensamt underhåll & administration** | Städning, reparation, logistik, mötesförberedelser | Veckovis/månatlig, flexibel arbetsnivå |

---

## 3. Veckovis rotation

- Varje medlem ansvarar **för minst 1 huvudområde per vecka**  
- Uppgifter kan kombineras (t.ex. Odling + Administration) för korta perioder  
- M-OS-R-appen visar **öppet schema** → alla ser vem som ansvarar för vad  
- Medlemmar med begränsad fysisk kapacitet kan välja **social eller administrativ roll**

### 3.1 Exempel på rotation (Vecka 1)

| Medlem | Odling & Skörd | Ocean-system | Refugium Anima | Underhåll/Admin |
|--------|----------------|--------------|----------------|----------------|
| M1     | Kort           | -            | -              | Veckovis       |
| M2     | Medium         | -            | -              | -              |
| M3     | -              | Kort         | -              | -              |
| M4     | -              | Medium       | -              | Veckovis       |
| M5     | Kort           | -            | Värd           | -              |
| M6     | -              | Lång         | -              | Kort           |
| M7     | Medium         | -            | -              | -              |
| …      | …              | …            | …              | …              |
| M29    | -              | Kort         | Värd           | Kort           |

> Notera: "-" betyder att medlem inte har ansvar i den uppgiften just denna vecka.  

Rotation sker automatiskt i M-OS-R varje vecka, men medlemmar kan **byta uppgifter efter dialog**.  

---

## 4. Anpassning för funktionsvariation

- Medlemmar med begränsad fysisk kapacitet får:  
  - Kortare pass  
  - Lättare uppgifter  
  - Administrativa eller sociala roller  

- Alla arbetsuppgifter är **delbara** (t.ex. skörd kan delas på 2 personer)  
- Transparens: schema visar ansvar, men **ingen övervakning av prestation**  

---

## 5. Poäng och erkännande (SRS)

- SRS används **endast för extraordinära bidrag** (inte Baseline-uppgifter)  
- Exempel:  
  - Innovativa odlingsmetoder  
  - Större reparationer, krishantering  
  - Kreativa workshops eller projekt  
- Erkännande: cirkulerande, temporärt, frivilligt  
- SRS påverkar **inte rotation eller Baseline-ansvar**

---

## 6. Målsättning

- Rättvis fördelning av alla uppgifter  
- Flexibilitet för olika förmågor  
- Öppenhet och transparens  
- Integrerat erkännande utan att påverka Baseline  

> Schemat ska göra det lätt för alla medlemmar att bidra, känna sig delaktiga och hålla HELIX gemenskapen livfull och trygg.
