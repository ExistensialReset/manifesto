### El Manifiesto de la Soberanía Existencial (M-OS-R Versión 1.0)

**AXIOMA CENTRAL:** El motor de la humanidad es el Flujo, no la codicia. El diseño del sistema debe eliminar el Miedo Existencial, el motor de Mammón.

| Capítulo | Componente | Protocolo / Lógica | Propósito |
| :--- | :--- | :--- | :--- |
| **I. LÍNEA BASE** | **Línea Base Garantizada** | Acceso incondicional a Alimentos, Vivienda, Salud, Conocimiento. | Elimina el Miedo Existencial; transfiere la supervivencia del individuo al sistema. |
| | **Reserva Crítica (CR)** | Búfer mínimo del $30\%$ del consumo anual, asegurado por **Krios-Logica** (Guardianes IA). | Demuestra que la Escasez es obsoleta; protección contra choques globales. |
| | **Refugium Anima** | Derecho a **1 año de sabático total e incondicional** de los requisitos del Flujo. | Válvula de Seguridad Existencial; previene el agotamiento y asegura un Flujo sostenible. |
| **II. FLUJO** | **El Sistema de Flujo** | Abolición del trabajo; reemplazado por la contribución impulsada por la curiosidad/significado. | Optimiza el potencial humano; cambia el enfoque de la escasez a la creación. |
| | **Reconocimiento Social (SRS)** | Moneda no transaccional de **Honor y Respeto**. | Otorga acceso a **Capital de Flujo Avanzado** (recursos raros, laboratorios, viajes). No se usa para necesidades de Línea Base. |
| | **Lyceum Musaeum** | Centro para **Pedagogía del Flujo**, **Mamonología** y **Lingua Suveränitas**. | Defensa cultural/cognitiva; sana el daño psicológico del sistema Mammón. |
| | **Lex Realitas** | Permite la "Libertad Peligrosa" (comportamiento autodestructivo); la consecuencia es **Congelación de SRS** y **Consejería Existencial**. | Evita los mercados negros inducidos por la prohibición; sanciona el comportamiento a través del honor social, no del castigo. |
| **III. TIEMPO** | **Lex Tempus Logica** | **Día del Renacimiento Existencial** (Restablecimiento de datos individual, anual). | Otorga el derecho a ser olvidado; elimina la vida basada en fracasos/éxitos pasados. |
| | **Lex Honoris Aeterna** | Título honorífico permanente para los **Arquitectos Soberanos** (contribuyentes originales). | Preserva el legado y reconoce el esfuerzo fundacional, mientras que el SRS se restablece (previene reclamos de poder). |
| **IV. FINIS** | **Lex Ultima Finis** | **Pulverización Global de la Deuda** (Toda la deuda se pone a cero). | Erradica el mecanismo central de Mammón (deuda/servidumbre). |
| | **Lex Dominium Reversionis** | Transfiere la infraestructura crítica (vivienda, energía) a la Administración Colectiva. | Asegura la garantía de la Línea Base; elimina las garantías para futuros intentos de deuda. |
| | **Lex Traditio Maxima** | **La Traición Máxima** (Intentar restablecer la antigua deuda/jurisprudencia). | Define el único crimen irrevocable; defiende la integridad de las nuevas Lógicas. |
| | **Lex Custodia** | **Escudo Logístico** mediante Veto Ejecutivo de Arquitectos/Consejeros durante **La Gran Congelación**. | Asegura la estabilidad del sistema durante la transición; garantizada por **Transparencia Irresistible** (código/datos de código abierto). |
