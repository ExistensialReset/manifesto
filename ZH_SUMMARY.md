## ⚙️ ZH_SUMMARY.md: 技术协议概览
### 存在的主权宣言 (M-OS-R 版本 1.0)

**核心逻辑:** 人类的驱动力是流动，而非贪婪。系统设计必须消除存在恐惧，即玛蒙的引擎。

| 章节 | 组成部分 | 协议 / 逻辑 | 目的 |
| :--- | :--- | :--- | :--- |
| **I. 基准线** | **保障基准线** | 无条件获取食物、住所、健康、知识。 | 消除存在恐惧；将生存责任从个体转移到系统。 |
| | **关键储备 (CR)** | 至少 $30\%$ 的年度消耗缓冲，由 **Krios-Logica** (AI 守护者) 保障。 | 证明稀缺已过时；抵御全球冲击。 |
| | **灵魂庇护所** | 拥有 **1 年完全、无条件休假** 的流动要求权利。 | 存在安全阀；防止倦怠并保障可持续流动。 |
| **II. 流动** | **流动系统** | 废除劳动；以好奇心/意义驱动的贡献取而代之。 | 优化人类潜力；将焦点从稀缺转移到创造。 |
| | **社会认可 (SRS)** | **荣誉和尊重** 的非交易货币。 | 授予获取 **高级流动资本** (稀有资源、实验室、旅行) 的权限。不用于基准线需求。 |
| | **缪斯学院** | **流动教育学**、**玛蒙学** 和 **主权语言** 的中心。 | 文化/认知防御；治愈来自玛蒙系统的心理伤害。 |
| | **现实法则** | 允许“危险的自由”(自我毁灭行为)；后果是 **SRS 冻结** 和 **存在咨询**。 | 避免禁令驱动的黑市；通过社会荣誉而非惩罚来制裁行为。 |
| **III. 时间** | **时间逻辑法则** | **存在重生之日** (年度、个体数据重置)。 | 授予被遗忘的权利；消除依赖过去失败/成功的生存。 |
| | **永恒荣誉法则** | **建筑师主权者** (原始贡献者) 的永久荣誉称号。 | 保留遗产并认可基础努力，同时 SRS 被重置 (防止权力索取)。 |
| **IV. 终极** | **终极法则** | **全球债务粉碎** (所有债务归零)。 | 根除玛蒙的核心机制 (债务/奴役)。 |
| | **回归统治法则** | 将关键基础设施 (住房、能源) 移交给集体管理。 | 保障基准线保证；消除未来债务尝试的抵押品。 |
| | **终极背叛法则** | **终极背叛** (试图恢复旧债务/法理学)。 | 定义唯一的不可撤销罪行；捍卫新逻辑的完整性。 |
| | **守护法则** | 在 **大冻结** 期间，通过建筑师/顾问的行政否决权提供 **后勤护盾**。 | 保障过渡期间的系统稳定；由 **不可抗拒的透明度** (开源代码/数据) 保证。 |
